# Gantt-chart-creator

## Sobre o Projeto
O **Gantt Chart Creator** é um aplicativo web interativo criado para facilitar a organização e visualização de etapas de projetos. Ele permite aos usuários criarem Diagramas de Gantt de forma simples e intuitiva, adicionando tarefas, definindo responsáveis, datas de início e fim, além de organizar as fases do projeto.

## Funcionalidades
- **Interface Dinâmica:** Adicione, edite e remova tarefas e fases diretamente no seu navegador de forma super rápida.
- **Visualização Clara:** Acompanhe o progresso do projeto com um diagrama visualmente organizado e fácil de entender.
- **Exportação em PNG:** Exporte seu Diagrama de Gantt completo em formato imagem (PNG) para fácil compartilhamento em relatórios, documentações ou apresentações.
- **Armazenamento Local:** Seus dados e o estado do projeto atual são salvos automaticamente no *localStorage* do seu navegador. 

## Tecnologias Utilizadas
- **HTML, CSS e JavaScript** 
- **[Alpine.js](https://alpinejs.dev/)**: Para o controle de estado e reatividade inteligente da interface.
- **[Flowbite](https://flowbite.com/)**: Biblioteca de componentes CSS utilizada para estilização moderna e elegante.
- **[SnapDOM](https://github.com/zumer/snapdom)**: Plugin em JavaScript responsável por capturar o visual do diagrama (DOM) e salvá-lo como imagem (PNG).

## Como Usar
Para usar a aplicação, basta abrir o arquivo `index.html` em qualquer navegador web moderno. 
Não há necessidade de processos de build complexos ou a instalação de um servidor de backend.
